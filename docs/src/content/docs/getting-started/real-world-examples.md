Looking for practical runok configurations? These repositories show how runok is used in real projects:

- **[fohte/dotfiles `runok.yml`](https://github.com/fohte/dotfiles/blob/main/config/runok/runok.yml)** -- A personal dotfiles configuration with rules for everyday shell commands.
- **[fohte/runok-presets](https://github.com/fohte/runok-presets)** -- The official preset collection. Reusable rule sets for common tools (Unix commands, Git, GitHub CLI, etc.). See [Official Presets](/configuration/official-presets/) for documentation.
