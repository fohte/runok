Released on 2026-03-11. [Full changelog](https://github.com/fohte/runok/compare/v0.1.3...v0.1.4)

## New Features

- `runok init` now warns when other PreToolUse hooks conflict with the runok sandbox ([#186](https://github.com/fohte/runok/pull/186))
- Literal tokens in patterns are now matched in an order-independent manner, so `git add .` matches a rule for `git add` regardless of argument order ([#177](https://github.com/fohte/runok/pull/177))

## Bug Fixes

- Fix flag-only negation patterns not passing when the command has no tokens ([#179](https://github.com/fohte/runok/pull/179))
- Fix `FlagWithValue` patterns not matching `=`-joined tokens (e.g., `--output=json`) ([#180](https://github.com/fohte/runok/pull/180))
- Fix flag-only negation patterns not matching `=`-joined tokens ([#175](https://github.com/fohte/runok/pull/175))
- Fix unmatched sub-commands in compound evaluation not resolving to concrete actions ([#178](https://github.com/fohte/runok/pull/178))
