Released on 2026-03-09. [Full changelog](https://github.com/fohte/runok/compare/v0.1.2...v0.1.3)

## New Features

- Relative paths in config files are now resolved based on the config file's parent directory, not the working directory ([#166](https://github.com/fohte/runok/pull/166))
- Add file-based locking for concurrent preset cache access, preventing corruption when multiple runok instances run in parallel ([#168](https://github.com/fohte/runok/pull/168))
- Support path specification in GitHub shorthand references for presets (e.g., `owner/repo:path/to/preset.yml`) ([#167](https://github.com/fohte/runok/pull/167))

## Bug Fixes

- Fix preset rules referenced via `extends` not being resolved correctly ([#174](https://github.com/fohte/runok/pull/174))
- Fix config file discovery not traversing ancestor directories ([#165](https://github.com/fohte/runok/pull/165))
- Fix order-independent matching for flag-only negation patterns ([#171](https://github.com/fohte/runok/pull/171))
