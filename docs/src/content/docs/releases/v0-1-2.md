## New Features

- Add `runok init` subcommand for interactive setup of runok configuration and Claude Code hook integration ([#152](https://github.com/fohte/runok/pull/152))

## Bug Fixes

- Fix command substitutions nested inside quoted strings not being evaluated correctly ([#151](https://github.com/fohte/runok/pull/151))
- Fix stack overflow in compound command evaluation when commands contain command substitutions ([#149](https://github.com/fohte/runok/pull/149))
- Fix flag alternation patterns consuming placeholder values as flag values ([#154](https://github.com/fohte/runok/pull/154))
- Treat `--` as a positional literal instead of a flag, matching standard Unix convention ([#153](https://github.com/fohte/runok/pull/153))
