## Documentation

### Surface official presets and Claude Code plugin in getting-started docs

The README and Quick Start now show `extends` with the [`base` preset](/configuration/official-presets/) in their default configuration example, so new users start with a curated read-only baseline instead of writing every rule from scratch. The README's Quick Start gained a dedicated section for installing the [Claude Code plugin](/getting-started/claude-code/#claude-code-plugin), and the Claude Code Integration page surfaces the plugin at the top so it is discoverable without scrolling to the end.
